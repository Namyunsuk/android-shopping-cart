# android-shopping-cart

---
### 기능 목록

- [ ] 상품 목록 화면
  - [x] 상품이 보인다
    - [x] 상품 이미지, 이름, 가격
    - [ ] 클릭 시 상품 상세 화면으로 이동한다.
  - [ ] 장바구니 버튼이 있다.
    - 클릭 시 장바구니 화면으로 이동  
    
    
    
- [ ] 상품 상세 화면
  - [ ] 상품이 보인다.
    - [ ] 상품 이미지, 이름, 가격
  - [ ] '장바구니 담기' 버튼이 보인다.  
  
    
- [ ] 장바구니 화면
  - [ ] 담은 상품이 보인다.
    - [ ] 상품 이미지, 이름, 가격
    - [ ] 'x' 버튼으로 상품을 지울 수 있다.
